## Windows, MacOS, Linux

- After cloning the repo
```bash
    cd <repo_name>
    pip3 install pygame
```
- If don't want to install Libs.... globally, you can create a virtual environment(recommended)
```bash
    cd <repo_name>
    python3 -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    pip3 install pygame
```

- Run the game
```bash
    python3 main.py
```
